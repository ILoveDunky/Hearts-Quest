# **App Name**: Hearts Quest

## Core Features:

- Start Screen: Display a message and a 'Start' button to begin the scavenger hunt.
- Profile Picture Memory Game: Asks user what was the first profile picture we matched and accepts 'frogs' and all similar variations of frogs as correct.
- Minecraft Memory Game: Asks user 'What was the first game we played together', and accepts 'Minecraft'.
- First Message Memory Game: Asks user 'What was the first thing I said to you when we first met', and accepts 'Who is you' and similar variations.
- Content Suggestion: Use generative AI tool that comes up with additional question topics that suit an anniversary between people. Store locally or in database as JSON object with parameters and descriptions. 
- Anniversary progress tracking: Have data model for questions and progress made. The user must successfully complete questions. As the user successfully answers the questions. This could increase a simple progress bar as they go.

## Style Guidelines:

- Primary color: Light purple (#D8B4FE) to create a romantic and playful vibe.
- Background color: Black (#000000) for a dramatic, intimate, and modern feel with a low saturated desaturation 20%.
- Accent color: Lavender (#E6E6FA) for buttons and interactive elements. This provides enough of a change in brightness and saturation, creating separation.
- Body and headline font: 'Alegreya', a transitional serif font that will provide a modern feel.
- Use heart icons and related elements for a romantic touch.
- Subtle star field animation in the background for visual appeal. Animation for questions lighting up and the transition animation after completion of task.
